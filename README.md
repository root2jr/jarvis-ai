Jarvis Web is a full-stack personal AI assistant built as a web application, focusing on task management, persistent memory, reminders, and AI-driven interactions. The project is designed around real-world assistant workflows rather than simple chat interfaces, with secure authentication, backend-controlled logic, and structured data handling.

The system emphasizes clean architecture, scalability, and reliability, serving as the web interface of the larger Jarvis ecosystem alongside its mobile counterpart.

Tech Stack

Frontend: React, TypeScript

Backend: Node.js, Express / FastAPI

Database: MongoDB

Auth: JWT, hashed passwords

AI: Integrated conversational logic

Machine Learning: For intent classification

Other: REST APIs, notification workflows

Core Features

Secure user authentication

Persistent user memory and conversation context

Task and reminder management

Automated notifications

AI-assisted interactions backed by server logic
